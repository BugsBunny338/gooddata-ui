AttributeElements Component

This is a low level counterpart to the AttributeFilter component that lists attribute values and lets you completely customize how they are rendered. The children function receives the following arguments:

  • An array of attributes
  • A callback to load more values
  • Loading status
  • An optional error object

Example

In the following example, attribute values are listed as a combination of checkboxes and label elements, a 'load more' button, and an onChange callback that logs the selected values to the console.

    // eslint-disable-next-line no-console

    const { title, uri } = this.props;

            <span>{title}</span>
        </label>
    );
}
