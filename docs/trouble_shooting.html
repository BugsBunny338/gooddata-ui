Troubleshooting

A table is broken or not rendered properly. The legend is missing colors.

Include CSS files in your application. For more information, see How to Create Your First Visualization with GoodData UI SDK, Step 7.

A chart legend is broken or positioned incorrectly.

Wrap all your visualization components (except KPIs) with the element with specified dimensions (height and width). For more information, see React Components.
